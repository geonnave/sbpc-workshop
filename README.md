# Plataformas Abertas para IoT

Projetos práticos para demonstração das plataformas Arduino e Labrador.

## Arduino

Plataforma aberta criada na Itália.

Demonstrações de:

* Led piscante
* Controle do led via serial (também possível via serial Bluetooth)

## Labrador

Plataforma aberta criada no Brasil. Pode ser usada como ambiente para programação do Arduino.

Demonstrações de:

* Led piscante
* Controle do led via Wi-Fi
* Detecção de faces
* Detecção de faces via Wi-Fi
* Controle do led sujeito à detecção de faces via Wi-Fi
